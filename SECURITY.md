# Politique de Sécurité

## Versions supportées

Seules les versions suivantes de Nocturne reçoivent des mises à jour de sécurité.

| Version | Supportée |
| ------- | --------- |
| 1.0.x   | Oui       |
| < 1.0   | Non       |

## Signaler une faille de sécurité

Si vous découvrez une vulnérabilité critique ou un moyen de faire crasher un serveur avec Nocturne, merci de ne pas créer une Issue publique. Veuillez la signaler en privé en envoyant un message à **Najom** sur discord.

Nous répondrons dans les plus brefs délais pour corriger le problème avant de le rendre public.
