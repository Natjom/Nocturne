# Nocturne
Projet de jeu de deduction sociale à roles cachés pour Minecraft (NeoForge 26.1).
L'objectif est de proposer une experience de jeu rapide et automatisee ou chaque joueur recoit un role unique avec des capacites specifiques durant une phase de nuit, dans le genre du Loup Garou de thiercelieux.

## Setup technique :
1. Preparer l'environnement (sous IntelliJ) :

`./gradlew genIntellijRuns`

2. Lancer le client :

`./gradlew runClient`
